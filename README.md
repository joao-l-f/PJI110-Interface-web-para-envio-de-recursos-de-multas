# PJI110---Interface-web-para-interposi-o-de-recursos-de-multas
Aplicação web em Flask e PostgreSQL para envio de recursos contra multas de trânsito à Secretaria de São Miguel Arcanjo, com foco em acessibilidade, segurança e usabilidade. Projeto em grupo conforme exigências da disciplina DRP05-Projeto Integrador em Computação I (UNIVESP).
